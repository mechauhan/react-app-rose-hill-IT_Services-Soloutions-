To configure nodemon in your node js application 


go to package.json

ADD there 2 command "start":"node index.js" AND "devStart":"nodemon index.js",

"scripts": {
    "start":"node index.js",
    "devStart":"nodemon index.js",


To run nodemon go to command prompt

npm run devStart
